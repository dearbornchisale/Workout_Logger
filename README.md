# 🏋️ Workout Logger

A modern web application for tracking gym workouts, exercises, and progress over time.

## 📖 About

Workout Logger is a personal fitness tracking application that allows users to:

- Create workout sessions
- Log exercises performed during each workout
- Record sets, reps, and weight
- View workout history
- Track strength progression over time

This project is being built as a full-stack portfolio project using modern web technologies.

---

## 🚀 Tech Stack

### Frontend
- Next.js
- React
- TypeScript
- Tailwind CSS
- shadcn/ui

### Backend
- Supabase
- PostgreSQL
- Supabase Authentication

### Deployment
- Vercel

---

## ✨ Features

- User authentication
- Create workout sessions
- Add exercises
- Log sets, reps and weight
- View workout history
- Edit workouts
- Delete workouts
- Responsive design

---

## 🗄️ Database

The application uses a relational PostgreSQL database with the following main tables:

- Users
- Workouts
- Exercises
- Workout Sets
